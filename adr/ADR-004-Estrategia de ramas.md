# ADR-004: Estrategia de versionamiento

## Contexto
Se requiere trabajo colaborativo.

## Problema
No hay estrategia de ramas.

## Decisión
Usar:
- main → producción
- qa → pruebas
- develop → desarrollo

## Justificación
Separación clara de entornos.

## Consecuencias
+ Mejor control
+ Flujo ordenado
